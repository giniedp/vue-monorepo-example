# Folder Structure

```
/
├── apps/                       # Folder with application packages
│   ├── hello-app/              # Demo vue application
│   └── .eslintrc.js            # eslint rules for apps
├── libs/                       # Folder with library packages
│   ├── hello-lib/              # Demo vue library
│   └── .eslintrc.js            # eslint rules for libs
├── tools/                      # development tools and scripts
│   └── .eslintrc.js            # eslint rules for tools
├── .editorconfig               # editor hints of how to format certain file types
├── tsconfig.base.json          # shared tsconfig rules
└── vetur.config.js             # vetur extension configuration
```
